# 🎓 UniceFight

UniceFight é um jogo 2D de luta desenvolvido como projeto acadêmico, inspirado na rotina de estudantes do curso de Análise e Desenvolvimento de Sistemas. No jogo, o jogador enfrenta professores que se tornam chefes, cada um representando uma disciplina da graduação e possuindo habilidades exclusivas baseadas em suas disciplinas e personalidades.

O projeto foi desenvolvido utilizando **JavaScript** e **HTML5 Canvas**, sem o uso de engines de jogos, permitindo a implementação manual de toda a lógica de movimentação, combate, inteligência artificial, colisões, animações e gerenciamento de estados.

---

## 📖 Sobre o projeto

A proposta do UniceFight é transformar experiências vividas durante a graduação em uma aventura divertida e desafiadora. Ao longo da jornada, o jogador precisa dominar diferentes padrões de ataque, aprender as fraquezas de cada chefe e adaptar sua estratégia para superar desafios cada vez mais complexos.

---

## ✨ Funcionalidades

* Sistema de combate em tempo real
* Movimentação lateral e salto
* Ataques corpo a corpo
* Sistema de defesa
* Barra de vida
* Inteligência artificial personalizada para cada chefe
* Mecânicas exclusivas em cada batalha
* Sistema de diálogos durante o combate
* Animações por spritesheet
* Sistema de progressão por fases
* Balanceamento individual de cada chefe

---

## 🛠️ Tecnologias utilizadas

* JavaScript (ES6+)
* HTML5
* CSS3
* HTML5 Canvas

---

## 🎯 Objetivos do projeto

* Aplicar conceitos de programação na prática
* Desenvolver um jogo sem utilização de engines
* Implementar inteligência artificial para NPCs
* Criar mecânicas de combate e progressão
* Exercitar lógica, organização e arquitetura de código
* Transformar experiências acadêmicas em uma experiência interativa
